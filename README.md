Небольшой проект "Панель управления задачами" с Material Design интерфейсом на Vue.js, Vuetify, Vuex и vue-i18n


ДОРАБОТКА ПРОЕКТА

В проект добавлен vue-i18n 9 для интернационализации (поддержка 8 языков).

Добавленный функционал:
1) Поддержка 8 языков: русский, английский, немецкий, французкий, белорусский, украинский, польский, казахский
2) Переключение языка через выпадающее меню в шапке (с флагами стран)
3) Выбор языка в настройках с отображением флагов
4) Сохранение выбранного языка в localStorage
5) Полный перевод всех текстов интерфейса: навигация, задачи, аналитика, команда, настройки, уведомления
6) Локализация уведомлений (создание, удаление, выполнение задач, смена языка, смена темы)
7) Динамическое обновление приоритетов задач при смене языка
8) Локализация дат в зависимости от выбранного языка

Структура локализации:
src/locales/
1) ru.js - русский язык
2) en.js - английский язык
3) de.js - немецкий язык
4) fr.js - французкий язык
5) be.js - белорусский язык
6) uk.js - украинский язык
7) pl.js - польский язык
8) kk.js - казахский язык

src/i18n.js - конфигурация vue-i18n с подключением всех языков

Обновления в модулях:

1) store/modules/settings.js
   - Добавлено поле locale в state
   - Добавлен getter currentLocale
   - Добавлена мутация SET_LOCALE
   - Добавлен action setLocale

2) views/Settings.vue
   - Добавлен выбор языка с флагами
   - Селект с кастомным отображением флагов

3) views/Tasks.vue
   - Приоритеты задач обновляются динамически при смене языка
   - priorityOptions переведен в computed свойство

4) App.vue
   - Добавлено выпадающее меню выбора языка в шапке с флагами
   - Динамическое отображение текущего языка (RU, EN, DE и т.д.)
   - Синхронизация языка между store и i18n

Технологии:
1) Vue.js 3
2) Vuetify 3
3) Vue Router 4
4) Vuex 4
5) vue-i18n 9
6) Vite
7) MDI Icons
8) flag-icons

Особенности реализации с vue-i18n:
1) Все тексты интерфейса вынесены в отдельные файлы локализации
2) Поддержка 8 языков с возможностью легкого добавления новых
3) Сохранение выбранного языка в localStorage
4) Уведомления отображаются на текущем выбранном языке
5) Приоритеты задач в диалоге создания обновляются мгновенно при смене языка
6) Выбор языка доступен как в шапке (с флагами), так и в настройках
7) Флаги стран отображаются для визуального удобства выбора языка

Команды для запуска.

Установка зависимостей:
npm install

Запуск в режиме разработки:
npm run dev

Сборка для production:
npm run build
